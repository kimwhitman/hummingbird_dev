# Class Dojo UX Overview

## 1. Parent Experience

**Bottom Tab Bar:**
`Kids | Chats | Notifications | Settings* | *avatar / "More"`

**Top contextual tabs (within Home):**
`Story Memories Calendar`

| Level 1 (tab) | Level 2 (section) | Key Elements / Tasks                                            |
| :------------ | :---------------- | :-------------------------------------------------------------- |
| **Kids**      | Child Switcher    | Add child (+), toggle multiple students [help.classdojo.com](https://help.classdojo.com) |
|               | Story Feed        | Class Story & School Story posts (cards → details)              |
|               | Portfolio         | Grid of student uploads, approve/comment, download              |
|               | Reports           | Points graphs, skills breakdowns, weekly email toggle           |
|               | Dojo Islands      | Home play zone (game) entry                                     |
| **Chats**     | Thread List       | 1-to-1 teacher chat, co-parent chat, class/family group chats, new chat (+) [help.classdojo.com](https://help.classdojo.com) |
|               | Thread View       | Translate, emoji, attachments, read receipts, block/unblock [help.classdojo.com](https://help.classdojo.com) |
| **Notifications** | Feed          | Story likes/comments, new messages, point awards, event reminders [help.classdojo.com](https://help.classdojo.com) |
| **Settings / More** |           | Profile, language, privacy/terms, ClassDojo Plus paywall, help/FAQ |

## 2. Teacher (and School-Leader) Experience — new iOS layout

**Bottom Tab Bar:**
`Classes | School | Chats | Notifications | + |`

**Top contextual tabs (within Classes & School):**
`Story Students · Points · Calendar · Directory, etc.`

| Level 1 (tab) | Level 2             | Level 3 / Utilities                                       | Notes                                                     |
| :------------ | :------------------ | :-------------------------------------------------------- | :-------------------------------------------------------- |
| **Classes**   | Class List          | Create/Archive class                                      | Multi-class jump list                                     |
|               | Inside a Class      | Story, Students grid, Portfolios, Activities, Reports, Dojo Islands | "Options" for Edit Class, Skills, Parents list            |
|               | Teacher Toolkit     | Timer, Randomizer, Music, Noise meter, Directions        | Floating toolbox                                          |
| **School**    | Directory (top tab) | Universal search (students/teachers/classes)              | [help.classdojo.com](https://help.classdojo.com)          |
|               | Calendar (top tab)  | Events, filter by class or whole school                   |                                                           |
|               | Points (top tab)    | School-wide point dashboard, bulk award                   | [help.classdojo.com](https://help.classdojo.com)          |
|               | School Story        | Feed, post composer, moderate comments                    |                                                           |
|               | Admin Tools         | Invite teachers, graduate/archive students, school settings |                                                           |
| **Chats**     | Unified inbox       | Filters: Unread / By Class / Staff                        | Group chat (+), staff chat, parent chat ([help.classdojo.com](https://help.classdojo.com), [help.classdojo.com](https://help.classdojo.com)) |

## 3. Admin / School-Leader Experience

**Bottom Tab Bar:**
`| Classes | School | Chats | Notifications | + |`

**Top contextual tabs inside **School**:**
`Insights · Story · Calendar · Points · Directory · Settings`

| Level 1 (tab)   | Level 2 (section) | Level 3 / Utilities                                       | Notes                                                     |
| :-------------- | :---------------- | :-------------------------------------------------------- | :-------------------------------------------------------- |
| **Classes**     | Class List        | Create / Archive / Join (add self as co-teacher)          | Lets admins hop into any class for moderation or support ([help.classdojo.com](https://help.classdojo.com)) |
|                 | Inside a Class    | View Story, Students, Reports, Skills; Edit Class; Reset QR codes | Same canvas as teachers, but with full read-only or edit rights based on permission set ([help.classdojo.com](https://help.classdojo.com)) |
| **School**      | Insights          | Engagement cards: msgs sent, photos shared, % positive feedback | Real-time health of school community ([help.classdojo.com](https://help.classdojo.com)) |
|                 | School Story      | Feed, post composer, comment moderation                   |                                                           |
|                 | Calendar          | Events (filter by grade / class)                          |                                                           |
|                 | Points Dashboard  | School-wide totals, bulk award, **Set Skills** (create/edit/delete) | Skills auto-sync to all classes ([help.classdojo.com](https://help.classdojo.com), [help.classdojo.com](https://help.classdojo.com)) |
| **Chats**       | Unified inbox     | Filters: Staff-only, Broadcasts, Family messages, Unread  | Admins can "broadcast to all" or step into any thread     |
| **Notifications** | All alerts    | Deep-links back to Insights, Directory, etc.              |                                                           |
| **Universal “+”** | Quick actions   | • Post (School/Class Story) • Message/Broadcast • Event • Award Points • Create Class | Visible from every tab                                    |
| **Directory**   | Students Staff Classes top tabs | Bulk import/graduate, merge duplicates, invite staff/parents | Central roster & SIS sync ([help.classdojo.com](https://help.classdojo.com), [help.classdojo.com](https://help.classdojo.com)) |
| **Reports**     |                   | Export CSVs (school-wide points, engagement)              | For PBIS, MTSS, board reports, etc. ([help.classdojo.com](https://help.classdojo.com)) |
| **Settings**    |                   | School logo, grade levels, privacy settings, district SSO/roster toggle | District controls propagate here when enabled ([techlearning.com](https://techlearning.com)) |
